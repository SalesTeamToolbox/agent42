# SEO Optimization

## On-Page SEO Checklist

### Title Tag
- Include primary keyword near the beginning
- Keep under 60 characters
- Make it compelling (not just keyword stuffing)
- Unique for every page

### Meta Description
- 120-160 characters
- Include primary keyword naturally
- Write as a compelling summary with CTA
- Unique for every page

### Heading Structure
- One H1 per page (include primary keyword)
- Use H2 for main sections (include secondary keywords)
- Use H3 for subsections
- Logical hierarchy (no skipping levels)

### Content Body
- Primary keyword in first 100 words
- Keyword density: 1-2% (natural, not forced)
- Use semantic variations and related terms (LSI keywords)
- Minimum 800 words for ranking potential (1500+ for competitive topics)
- Use short paragraphs (2-3 sentences max)
- Include bullet points and numbered lists

### Internal Linking
- Link to 2-5 related pages/posts
- Use descriptive anchor text (not "click here")
- Link from high-authority pages to new content
- Update old posts with links to new content

### Images
- Add descriptive alt text with keywords where natural
- Compress images for fast loading
- Use descriptive file names (not IMG_001.jpg)
- Include at least one relevant image per 300 words

### URL Structure
- Include primary keyword
- Keep short and readable
- Use hyphens between words
- Lowercase only

## Keyword Research Workflow
1. Start with seed keywords (your product/topic)
2. Expand with related terms and questions
3. Check search volume and competition
4. Prioritize: high volume + low competition = best opportunities
5. Group keywords by intent (informational, navigational, transactional)
6. Map keywords to content types (blog, landing page, product page)

## Content Optimization Tips
- Answer the search intent directly in the first paragraph
- Use FAQ sections to capture "People Also Ask" snippets
- Include structured data where appropriate (how-to, FAQ, product)
- Write for humans first, optimize for search second
- Update and refresh content regularly (every 6-12 months)

## Technical SEO Basics
- Page load speed under 3 seconds
- Mobile-friendly responsive design
- HTTPS/SSL certificate
- XML sitemap submitted to search engines
- robots.txt properly configured
- No broken links (404 errors)
- Canonical tags on duplicate content

## Content Types by SEO Value
- **Pillar pages** (2000+ words): Comprehensive topic coverage
- **Blog posts** (800-1500 words): Target long-tail keywords
- **How-to guides**: High search intent, featured snippet potential
- **Listicles**: Easy to scan, good for backlinks
- **Comparison posts**: Capture "vs" and "best" queries
- **FAQ pages**: Direct answers, voice search optimization
