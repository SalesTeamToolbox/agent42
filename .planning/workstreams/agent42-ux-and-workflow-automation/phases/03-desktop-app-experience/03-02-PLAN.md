Per D-10 through D-16, the subcommand must:

**1. Detect platform** using `uname -s`:
```bash
OS="$(uname -s)"
```

**2. Detect browser** (per D-14 — Chrome preferred, Edge fallback on Windows, Chromium on Linux):

For **Windows** (MINGW*, MSYS*, CYGWIN*, or `$OS` contains "NT"):
- Check for Chrome: `"/c/Program Files/Google/Chrome/Application/chrome.exe"` or `"/c/Program Files (x86)/Google/Chrome/Application/chrome.exe"`
- Check for Edge: `"/c/Program Files (x86)/Microsoft/Edge/Application/msedge.exe"`
- Store the found path in `BROWSER_PATH` and name in `BROWSER_NAME`

For **macOS** (`Darwin`):
- Check: `/Applications/Google Chrome.app/Contents/MacOS/Google Chrome`
- Fallback: Print warning that Safari doesn't support `--app` flag (per D-16), recommend installing Chrome

For **Linux**:
- Check: `command -v google-chrome` then `command -v google-chrome-stable` then `command -v chromium-browser` then `command -v chromium`
- Store found command in `BROWSER_CMD`

If no supported browser found on any platform, call `error "No Chromium-based browser found. Install Google Chrome: https://www.google.com/chrome/"`.

**3. Create shortcut based on platform:**

**Windows** (per D-11):
Create a `.lnk` file on the Desktop using PowerShell. The PowerShell command:
```bash
DESKTOP="$(cmd.exe /c 'echo %USERPROFILE%\\Desktop' 2>/dev/null | tr -d '\r')"
ICON_PATH="$(cygpath -w "$PROJECT_DIR/dashboard/frontend/dist/assets/icons/icon-512.png" 2>/dev/null || echo "$PROJECT_DIR\\dashboard\\frontend\\dist\\assets\\icons\\icon-512.png")"
powershell.exe -NoProfile -Command "
  \$ws = New-Object -ComObject WScript.Shell;
  \$sc = \$ws.CreateShortcut('$DESKTOP\\Agent42.lnk');
  \$sc.TargetPath = '$BROWSER_PATH';
  \$sc.Arguments = '--app=http://localhost:8000';
  \$sc.IconLocation = '$ICON_PATH';
  \$sc.Description = 'Agent42 - AI Agent Platform';
  \$sc.Save()
"
```
Print: `info "Shortcut created! Find Agent42 on your Desktop."`

**macOS** (per D-12):
Create a `.app` bundle at `$HOME/Applications/Agent42.app`:
```bash
APP_DIR="$HOME/Applications/Agent42.app/Contents/MacOS"
mkdir -p "$APP_DIR"
mkdir -p "$HOME/Applications/Agent42.app/Contents/Resources"

# Copy icon
cp "$PROJECT_DIR/dashboard/frontend/dist/assets/icons/icon-512.png" \
   "$HOME/Applications/Agent42.app/Contents/Resources/agent42.png"

# Create launcher script
cat > "$APP_DIR/Agent42" << 'LAUNCHER'
#!/bin/bash
open -a "Google Chrome" --args --app=http://localhost:8000
LAUNCHER
chmod +x "$APP_DIR/Agent42"

# Create Info.plist
cat > "$HOME/Applications/Agent42.app/Contents/Info.plist" << 'PLIST'
<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE plist PUBLIC "-//Apple//DTD PLIST 1.0//EN" "http://www.apple.com/DTDs/PropertyList-1.0.dtd">
<plist version="1.0">
<dict>
  <key>CFBundleName</key>
  <string>Agent42</string>
  <key>CFBundleDisplayName</key>
  <string>Agent42</string>
  <key>CFBundleIdentifier</key>
  <string>com.agent42.app</string>
  <key>CFBundleExecutable</key>
  <string>Agent42</string>
  <key>CFBundleVersion</key>
  <string>1.0</string>
  <key>CFBundlePackageType</key>
  <string>APPL</string>
</dict>
</plist>
PLIST
```
Print: `info "Shortcut created! Find Agent42 in ~/Applications/."`

**Linux** (per D-13):
Create a `.desktop` file at `~/.local/share/applications/agent42.desktop`:
```bash
DESKTOP_DIR="$HOME/.local/share/applications"
mkdir -p "$DESKTOP_DIR"
ICON_PATH="$PROJECT_DIR/dashboard/frontend/dist/assets/icons/icon-512.png"

cat > "$DESKTOP_DIR/agent42.desktop" << DESKTOP
[Desktop Entry]
Name=Agent42
Comment=AI Agent Platform — Don't Panic
Exec=$BROWSER_CMD --app=http://localhost:8000
Icon=$ICON_PATH
Type=Application
Categories=Development;
StartupWMClass=agent42
DESKTOP
chmod +x "$DESKTOP_DIR/agent42.desktop"
```
Print: `info "Shortcut created! Find Agent42 in your application launcher."`
