       Request schema `MockupRequest(BaseModel)`:
       - `design_id: str | None = None`
       - `canvas_data_url: str | None = None` (base64 PNG from canvas export)
       - `product_id: int | None = None`

       Implementation:
       - Requires auth (Depends get_current_user)
       - If design_id provided, look up the design to get image_url
       - If canvas_data_url provided, decode the base64 data-url, save to a temp file in designs_dir, use that as the image URL
       - If product_id provided, call `generate_mockup(image_url, product_id)` from image_pipeline
       - If no product_id, return the processed image URL directly (no Printful mockup without product)
       - Return `{"mockup_url": str, "design_id": str | None}`
       - On Printful failure (generate_mockup returns None), return 503 with detail "Mockup generation failed or Printful API key not configured"

    2. **POST /save** (DES-09 per D-15, D-16):
       Request schema `SaveDesignRequest(BaseModel)`:
       - `design_id: str`
       - `canvas_data_url: str | None = None`

       Implementation:
       - Requires auth
       - Look up design by design_id, verify it belongs to current_user
       - If canvas_data_url provided, decode and save as `{design_id}_canvas.png` in the user's designs directory
       - Update design status to 'saved' in the database:
         `UPDATE designs SET status = 'saved' WHERE id = ? AND user_id = ?`
       - Return `{"status": "saved", "design_id": str}`

    3. **POST /upload** (DES-08 per D-14):
       Accepts multipart form: `file: UploadFile`, no JSON body.
       Add `garment_type: str = Form("tshirt")` and `style_preset: str | None = Form(None)` as form fields.

       Implementation:
       - Requires auth
       - Validate file content type is in ["image/png", "image/jpeg", "image/svg+xml"]
       - Validate file size under 10MB (read content, check len)
       - Generate design_id = str(uuid.uuid4())
       - Save file to `{settings.designs_dir}/{user_id}/{design_id}.png`
       - Create designs dir if not exists: `Path(settings.designs_dir, str(user_id)).mkdir(parents=True, exist_ok=True)`
       - Write file with aiofiles
       - Insert row into designs table with: provider='upload', model='user-upload', image_url as the relative serve path (`/designs/{user_id}/{design_id}.png`), image_path as the absolute local path, generation_cost_usd=0.0, status='draft'
       - Also create a design_session if user doesn't have an active one (or use a dedicated "uploads" session)

                   """UPDATE designs

                       pipeline_result["total_cost_usd"],
                       design_id,

               )
               await db.commit()
           # Log pipeline cost
           from services.image_gen import GenerationResult as _GR, log_generation_cost
